## Contribution Guidelines

### For Bug Fixes:
1. Fork the repository
2. Create a branch: `git checkout -b fix/problem-b23-syntax`
3. Make your changes
4. Test the query against the schema
5. Submit a pull request with:
   - Problem number in the title
   - Description of the issue
   - Your tested solution

### For New Problems:
Before proposing new problems, ensure they:
- Use the existing schema (no new tables)
- Fit the difficulty level (Basics/Intermediate/Advanced)
- Solve a clear business question
- Don't duplicate existing problems

### Code Style:
- Use consistent formatting (see existing problems)
- Include inline comments for complex logic
- Use meaningful table aliases
- Follow PostgreSQL best practices

## Question or Need Help?

- **Learning SQL**: This repo is for practice, not tutoring. Try the problem first, then review solutions.
- **Schema Issues**: Open an issue if the schema doesn't load properly
- **Clarifications**: Ask questions in GitHub Discussions (if enabled) or open an issue
